# rs-soroban-sdk
Rust SDK for writing contracts for [Soroban].

Soroban: https://soroban.stellar.org

Docs: https://docs.rs/soroban-sdk

[Soroban]: https://soroban.stellar.org

## Support

Bug fixes are provided on top of the latest patch version of the latest major
version.

Security fixes are provided on top of the latest patch version of every major
version including and since v22.

All other versions are unsupported and do not receive bug or security fixes.

This policy may change and this document will be updated when it does.
